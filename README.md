# 📅 Бот «Організатор подій»

Консольний чат-бот на Python, що допомагає студенту планувати та відстежувати події: лекції, іспити, зустрічі та нагадування.

---

## Мета проєкту

Розробити зручного бота-помічника, який дозволяє:
- додавати, редагувати та видаляти події;
- переглядати події за датою, категорією або ключовим словом;
- отримувати нагадування про сьогоднішні, завтрашні та найближчі події;
- зберігати дані між сесіями у JSON-файлі.

---

## Алгоритм роботи програми

```
Запуск програми
    │
    ├─► Завантаження подій з events.json (або порожній список)
    │
    ├─► Вітання користувача
    │
    └─► Головний цикл:
            │
            ├─ Зчитати команду від користувача
            │
            ├─ Знайти команду у словнику commands{}
            │       │
            │       ├─ Виконати відповідну функцію
            │       │       (додати / показати / редагувати / ...)
            │       │
            │       └─ Якщо дані змінились → зберегти у events.json
            │
            └─ "вийти" → завершити програму
```

---

## Структура даних

Кожна подія — це словник Python, що зберігається у списку:

```python
{
    "name":     "Іспит з Python",   # назва події (str)
    "date":     "2025-06-10",        # дата у форматі YYYY-MM-DD (str)
    "time":     "09:00",             # час початку HH:MM (str)
    "category": "Іспити",           # категорія / опис (str)
    "duration": 120                  # тривалість у хвилинах (int | None)
}
```

Усі події зберігаються у файлі **`events.json`** у корені проєкту.  
При запуску бот зчитує цей файл; після кожної зміни автоматично перезаписує його.

**Структура файлу `events.json`:**
```json
[
  {
    "name": "Лекція з математики",
    "date": "2025-06-02",
    "time": "10:00",
    "category": "Лекції",
    "duration": 90
  }
]
```

## Опис функцій

| Функція | Призначення |
|---|---|
| `load_events()` | Зчитує події з `events.json` |
| `save_events(events)` | Записує список подій у `events.json` |
| `parse_date(text)` | Перетворює рядок → `date` |
| `parse_time(text)` | Перетворює рядок → `time` |
| `format_event(i, ev)` | Форматує подію для виводу |
| `check_conflict(events, new)` | Знаходить події, що перетинаються за часом |
| `cmd_greet()` | Вітання |
| `cmd_help()` | Список команд |
| `cmd_add_event(events)` | Додавання події з перевіркою конфліктів |
| `cmd_show_events(events)` | Показати всі події |
| `cmd_week_events(events)` | Події поточного тижня |
| `cmd_events_by_date(events)` | Фільтр за датою |
| `cmd_events_by_period(events)` | Фільтр за діапазоном дат |
| `cmd_events_by_category(events)` | Фільтр за категорією |
| `cmd_search(events)` | Пошук за ключовим словом |
| `cmd_today(events)` | Події на сьогодні |
| `cmd_tomorrow(events)` | Події на завтра |
| `cmd_nearest(events)` | Найближча майбутня подія |
| `cmd_edit_event(events)` | Редагування події |
| `cmd_delete_event(events)` | Видалення події |
| `main()` | Головний цикл програми |

## Інструкція зі запуску

**Вимоги:** Python 3.10 або новіший (використовується `date | None` у анотаціях).

```bash
# 1. Клонувати репозиторій
git clone https://github.com/<ваш-логін>/<назва-репо>.git
cd <назва-репо>

# 2. Запустити бота
python bot.py
```

Файл `events.json` буде створено автоматично після додавання першої події.
